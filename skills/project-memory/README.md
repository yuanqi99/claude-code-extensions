# 项目记忆技能 (Project Memory Skill)

[English](./README_EN.md)

这是一个基于 `MEMORY.md` 文件的轻量级、原生的项目记忆管理技能。它不需要安装任何数据库或复杂的依赖，直接利用 Claude Code 的技能机制来维护项目上下文。

## 功能特性

- **自动同步**：在会话开始时自动读取项目背景。
- **经验沉淀**：自动识别 2 次及以上出现的模式、偏好和错误，并记录到记忆文件中。
- **决策追踪**：记录重大技术决策及其背景。
- **零依赖**：仅通过 Prompt 指令实现，不占用额外系统资源。

## 使用指南

### 1. 安装到 Claude Code

你可以通过以下两种方式之一安装此技能：

**方法 A：项目级安装（推荐）**
将此文件夹复制到你项目的技能目录中：
```bash
mkdir -p .claude/skills
cp -r path/to/project-memory .claude/skills/
```

**方法 B：全局安装**
将其复制到 Claude Code 的全局技能目录（通常在 `~/.claude/skills/`）。

### 2. 激活与使用

- **自动激活**：当你提到“记住这个”、“更新项目背景”或“保存到记忆”时，Claude 会自动触发此技能。
- **手动触发**：你可以直接说：“使用项目记忆技能分析当前会话并更新 MEMORY.md”。

### 3. 记忆文件结构

该技能会自动在项目根目录维护一个 `MEMORY.md` 文件，包含以下部分：
- **项目概览**：核心目标。
- **技术栈**：使用的语言和框架。
- **约定规范**：代码风格、命名习惯。
- **重大决策**：技术决策记录。
- **经验教训**：避坑指南。

## 注意事项

- 只有当某个模式在 2 个或更多会话中出现时，该技能才会建议将其永久记入 `MEMORY.md`。
- 所有的修改都会先向你确认，确保记忆的准确性。
